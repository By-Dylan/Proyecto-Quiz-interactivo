Basados En Estos Principio


Innovacion

Creemos en una innovación que desafía lo convencional. Esta web ha sido creada bajo la promesa de expandir constantemente las fronteras de la tecnología educativa. ¿Nuestro objetivo? Ir un paso más allá para transformar la educación digital en una experiencia interactiva, inmersiva y sin límites.

Comunidad

Esta plataforma no nos pertenece a nosotros, les pertenece a las mentes inquietas que la habitan día a día. Tu voz es nuestra hoja de ruta. Aquí no eres un simple espectador, eres el arquitecto de tu propio entorno de aprendizaje.

Referencia hacia las APIs Públicas

Nombre: Open Trivia Database

Breve descripción: Es una base de datos gratuita de preguntas de trivia. Actúa como el motor principal de QuizMind, encargándose de generar aleatoriamente las preguntas, la respuesta correcta y las alternativas falsas para cada partida.

Tipo: API REST

Dificultad: Cuenta con tres niveles de dificultad: Easy, Medium y Hard.

Categorías: Cuenta con 24 temáticas diferentes para elegir, abarcando áreas como Ciencias de la Computación, Ciencia, Películas, Geografía, Música, Deportes, entre otras.

Formato .json de la API

Nombre: MyMemory Translated API

Breve descripción: Es un servicio de traducción colaborativo en tiempo real. En este poyecto se utiliza mediante el endpoint "/get" para recibir las preguntas de la API principal y traducirlas automáticamente de inglés a español para la interfaz del usuario.

Tipo: API REST - Endpoint GET

Dificultad: No aplica.

Categorías: No aplica.
